# Blobs and Device Tree files for the SM-G361F

This contains all device blobs and device tree files necessary for custom ROM development on the SM-G361F.

The SM-G361F is a model of the Samsung Galaxy Core Prime Value Edition and currently has barely any community support besides a custom TWRP version and this should hopefully make it alot easier to make ROMs for it.

If you are using this im guessing you already know how these kind of stuff works so i wont really include a guide or anything, if im a dumb idiot and made a mistake contact me via my [discord server](https://discord.gg/RePmuyn2VF)
